# NeteaseMusic-Unlocker

解锁网易云音乐客户端无版权以及VIP试听歌曲 BY NULL

## 通知！！！

由于作者开学，平时很少放假，所以项目暂停维护，春节假期再维护！不过欢迎各位网友自发维护，提出意见。（下一个版本更新界面，支持UPW,解决会覆盖之前的设置的BUG）

别催了，作者很努力的在写了，估计大概两天时间吧。

## 特性

- 使用其他软件音源替换无版权歌曲链接

- 本项目根据[nondanee](https://github.com/nondanee)的[UnblockNeteaseMusic](https://github.com/nondanee/UnblockNeteaseMusic)项目进行制作管理脚本

- 无需手动进行繁琐的配置，运行即用，自动配置

## 运行

1. [在这里](https://github.com/NULL204/NeteaseMusic-Unlocker/releases/)下载最新的版本（提供一键配置脚本下载）

2. 运行安装文件

3. 安装软件

4. 完成

## 注意！！！

暂支不支持UWP版

本软件为个人开发软件，没用给360等杀毒软件厂商交保护费，请自行添加信任！

## 已知错误

1. 无法设置音效

2. 无法使用第三方登录

> 修复方法：打开软件 => 进入管理界面 => 关闭网易云音乐代理 => 进行登录或设置音效操作 => 打开网易云音乐代理

3. 会覆盖之前的设置
